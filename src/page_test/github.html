关于本项目

在网页里选中内容后、高亮、批注、分享，以及扩展你需要的功能，如反馈，搜索等。

使用方式

npm install pagenote --save

import Pagenote from 'pagenote';
import 'pagenote/dist/pagenote.css';
var pagenote = new Pagenote('demos',option);
pagenote.init();
