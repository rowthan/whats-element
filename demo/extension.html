whats-element Chrome Extension

 chrome 扩展应用：获取网页元素信息

Download    JavaScript library      View on GitHub

使用方法：
  • 点击上方 Download 下载 chrome 扩展应用
  • chrome 地址栏输入： chrome://extensions/
  • 开启“开发者模式”。将下载的 whatsElement.zip 直接拖入浏览器中，完成安装。
  • 点击任意网页元素，右击选择“whats element”;或点击右上角whats-element logo；或使用快捷键 Ctrl+Shift+E。三种方式任意一种即可唤醒whats-element
