whats-element.js

 为每一个 HTML 元素计算唯一标识符(仅 3.9kb)

Download    Chrome 插件    GitHub

点击下列粉色内区域预览计算结果

元素本身具备 id 属性情况下：

id=singleIdid唯一 id=multiIdid不唯一时 id=multiId与前一个元素id相同 id=1 id为数字

元素具备 class 属性情况下：

class=card class=card 需要通过序号定位 class=card new 具备多个class

本身的所有属性都不能唯一定位时，需要借助父节点进行定位

class=a 1 class=a 2 class=a 3 不具备任何属性

具备 name 属性情况下：

radio 具备value属性时：

value="male"
value="female"
value 不唯一时

元素具备 name、class 但都不能唯一标识时

name="username" value="a"
name="username" value="b"
name="password" value="b"

HTML 中的元素可能具备 id,class,name,value,type 等，甚至不具备任何属性。或许多个 HTML 元素具备相同的属性，不具备唯一标识的条件。 无论是何种情况 whatsElement.js 都能够正确的运算出该 HTML 元素在该页面中的唯一标识符，计算出的结果，可通过一下三个原生API 获取到该元素。

  • document.getElementById()
  • document.getElementsByName()[0]
  • document.querySelector()

使用

      
        <script src="../dist/whatsElement.js"></script >
        <script>
           var whats = new whatsElement();
           document.addEventListener('click',function (event) {
             var result = whats.getUniqueId();//可选传入参数 event.target，当不传入参数时，默认计算最后一次鼠标点击到的HTML元素。
             var elementId = result.uniqueId;// 该元素的唯一标识符
             var queryType = result.queryType;// 定位该元素的类型，如 id,class,name...
           })
       </script>
